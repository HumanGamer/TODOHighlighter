# TODO Highlighter

Prefixed comment highlighting.

## Supported languages
- C
- C++
- C#
- Javascript
- TypeScript

## Supported Keywords
- TODO:
- FIXME:
- TEMP:
- TMP: 

## Supported Visual Studio Versions
- Visual Studio 2019
- Visual Studio 2022

Credit to: https://github.com/Benio101/Color.Comment for some of the code.
